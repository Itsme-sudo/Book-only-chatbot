Step 4: How to Use

Put your book content in book.txt. Each paragraph on a new line.

Run the assistant:

python assistant.py


Speak your question clearly. Example:

Who is the main character?
What is photosynthesis?
When did the event happen?


Say “exit” to quit.

How it Works

Offline voice recognition using Vosk

Embeddings for each paragraph of your book

Cosine similarity finds the paragraph closest in meaning to your question

Answers only from your book

BookAI/
 ├ assistant.py       # Main Python script
  ├ book.txt           # Your book / content file
   ├ model/             # Vosk offline speech model (download separately)
    ├ requirements.txt   # Python dependencies
     ├ setup.py           # Optional auto-setup script
      └ README.md          # Instructions
